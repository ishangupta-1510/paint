### ***Magic Paint🖌️🎨***
---
#### *This is a paint application built with MERN stack which has the functionality of login and sign up. It is integrated with MongoDB to store user info.*

---

#### ***Technologies used:***
> ***Frontend***
* ***ReactJs***

> ***Backend***
* ***NodeJs***
* ***ExpressJs***
* ***MongoDB***
